# Floppa

Floppa is a pateform made to let users sell their products and to make offers on other people's products.

## Prerequisites

- Java 11
- Maven

To test with database 
- MongoDB
- Docker 

## Commands

### Build

```
mvn clean install
```

### Execute

```
mvn exec:java
```
